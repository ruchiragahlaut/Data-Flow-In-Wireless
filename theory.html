Theory:

WSN (Wireless Sensor Network) contain number of sensors with the speciality to communicate among themselves and to a base station or sink node for different applications. a wireless sensor network has many sensor nodes that are interconnected by wireless communication channels.Sensor nodes is a small device that is used to monitor the physical and environmental condition such as temperature, sound and any movement in the sensing field. Sensor nodes senses the data and sends to the nearest sensor node and all sensor nodes co-operately pass their data to the base station and then data is accessed via internet or satellite by task manager node. Task manager node is basically end using, which want to monitor the particular area. Nowadays wireless sensor networks are popular and their applications are used by our military for environment monitoring.


now we will look for the wireless sensor protocol stack :



  • The physical layer is responsible for frequency selection, carrier frequency generation, signal detection, modulation, and data encryption.
  • The MAC layer is responsible for the multiplexing of data streams, data frame detection, medium access and error control. It ensures reliable point-to-point and point-to-multipoint connections in a communication network.
  • The network layer takes care of routing the data supplied by the transport layer.
  • The transport layer helps to maintain the flow of data if the sensor networks application requires it.In addition, the power, mobility, and task management planes monitor the power, movement, and task distribution among the sensor nodes. The power management plane manages how a sensor node uses its power.


Some useful points and limitations of data flow in wireless network :
  • It avoids lot of wiring.
  • It can be used in large scale and it is very flexible in environmental conditions.

  • It is very costly.
  • As we cannot control the flow of waves it is very easy to hack the information
